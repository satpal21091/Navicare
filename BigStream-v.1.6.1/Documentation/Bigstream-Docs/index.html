Getting Started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. /css — folder with css files.
  2. /fonts — folder with icon fonts.
  3. /images — folder with image files.
  4. /js — folder with Javascript files.
  5. /rs-plugin — folder with Slider Revolution files.
  6. /works — folder with Ajax projects files.

  7. intro.html — page with the list of template pages.
  8. index.html, index-01.html, index-02.html, index-03.html, ... — different variants of home page layout.
  9. blog-list.html, blog-post.html, blog-grid-1.html, blog-grid-2.html, blog-grid-3.html  — blog pages.
  10. pricing.html — pricing page.
  11. portfolio-page-1.html, portfolio-page-2.html, ... — extra portfolio pages.
  12. project-page-1.html, project-page-2.html, ... — single project pages.
  13. ui-elements.html — the list of  the elements which you can use on your website.
  14. ui-icons.html — the list of the icons which you can use on your website.
  15. ui-buttons.html — different button variants.
  16. ui-typography.html — grid and typography.

  17. contact_me.php — file with contact form settings.

Upload the template files (inside the folder www) to server with the help of one of the FTP-clients (for example, Total Commander).

Note: for the correct operation of all functions of the template, including AJAX methods (portfolio expander) and Contact form, you must upload the template to web-server.

Note:You can find contact form customization guide inside the section "Contacts Section".

Logotype

To change logotype in the dark navigation panel, find image images/logo-dark.png and replace it with your logo.

Note: to ensure Retina ready, the width and height of the image should be twice bigger than the values specified in the attributes width and height. By default, the tag’s attributes logo size is set 80x27, but the actual size of the image is 160x54px.

To change logotype in the white navigation panel, find image images/logo-white.png and replace it with your own logo.

Note: to ensure Retina ready, the width and height of the image should be twice bigger than the values specified in the attributes width and height. By default, the tag’s attributes logo size is set 80x27, but the actual size of the image is 160x54px.

    FAQ

    1. How to remove smooth scroll?
      Find and delete the line (in the end of the page):

    2. How to remove appear animation from page elements?
      Find and delete the class appear-animate form <body> tag.

    3. Why I have problems with opening of works’ descriptions in portfolio while local browsing?
      Ajax methods of Jquery are used for loading descriptions of the works and they can work only on the real server. For the correct operation of project expander, upload your website on a real server.

    4. Why contact form may doesn’t work?
      For the correct operation of contact form, the php support is necessary. Upload your website on a server with php support.

    5. Why there are no images that were in live demo?
      Envato forbids including demonstration images to templates’ files.
